## 🟦 Boju — Learning core (Home / Lessons / Reading / Review / Swipe / Saved)

### C1. WebP + AppImage device QA (эхэнд хий)

Өчигдөр бүх зураг WebP болж, remote зураг `AppImage` руу шилжсэн. Чиний
screen-үүд дээр зураг зөв ачаалж байгааг **төхөөрөмж дээр** шалга:

- **Screens:** `app/(tabs)/index.tsx` (home), `app/(tabs)/lessons.tsx`,
  `app/reading/index.tsx` + `app/reading/[id].tsx`, `app/swipe.tsx`,
  `app/saved.tsx`, `src/components/VocabCard.tsx`, `FlashCard.tsx`.
- **Шалгах:** бүх зураг гарч байна уу (хоосон/эвдэрсэн зураг байхгүй), background
  зурагнууд зөв, хурд сайжирсан эсэх.
- Хэрэв ямар нэг зураг гарахгүй бол: тухайн `require('...png')` үлдсэн эсэхийг
  шалга (бүгд `.webp` болсон байх ёстой), эсвэл remote зургийг `AppImage`-аар
  дүрсэл (`src/components/AppImage.tsx`).
- **Done:** бүх learning-core screen-д зураг зөв, jank багассан.

### C2. Light / Dark mode audit

Boju light/dark mode оруулсан (`useColors()` / `useSettings().palette`). Чиний
screen-үүд **light mode дээр** зөв харагдаж байгааг шалга:

- Static `colors`-ыг hardcode хийсэн газар бараан үлдэж болзошгүй →
  `useColors()`-оор солих. Шалгах файлууд: `reading/[id].tsx`, learning-core
  screen-үүд.
- **Тест:** Settings → Light mode → бүх screen тунгалаг/цайвар зөв, текст уншигдана.
- **Done:** learning-core бүх screen light + dark хоёуланд зөв.

### C3. (Stretch) Product-brief alignment (M6)

Цаг гарвал `docs/FUTURE_PLAN.md` + `ROADMAP.md` дэх M6: level / English-name /

## 🟩 Choi — Games / Social / AI (Quiz-Soril / AI chat / Leaderboard / Profile)

### B1. AI Buddy voice — device QA (эхэнд хий)

`app/(tabs)/chat.tsx` бүрэн дахин бичигдсэн (дуут яриа). Бодит төхөөрөмж дээр
турш (backend ажиллаж, `ANTHROPIC_API_KEY` + `ELEVENLABS_API_KEY` тохирсон
байх — **Өсөхбаяр-аас** авах):

- Buddy сонголт `GET /ai/buddies`-ээс ачаалагдана.
- **Push-to-talk:** микрофон дараад ярих → transcript + богино хариу + audio.
- **Correction card**, follow-up асуулт харагдана.
- **Usage meter** (минут), `VOICE_LIMIT` хүрэхэд текст горим руу шилжинэ.
- **Done:** нэг бүтэн дуут turn ажиллаж, audio тоглож байна.

### B2. AI Buddy "Санах ой" screen (шинэ ажил)

Backend API бэлэн (`src/api/ai.ts` → `getBuddyMemory`, `clearBuddyMemory`),
гэхдээ **UI хийгдээгүй**. Profile / settings-д нэм:

- "AI Buddy санах ой" мөр → жагсаалт (`GET /ai/buddy/memory`) + "Устгах"
  товч (`DELETE`, баталгаажуулах dialog).
- Файл: `app/(tabs)/profile.tsx` эсвэл `app/settings.tsx`; API аль хэдийн бий.
- **Done:** санах ойгоо харах + цэвэрлэх боломжтой.

### B3. Quiz / games post-merge QA

Өчигдрийн merge дараа чиний нэмсэн зүйлс зөв ажиллаж байгааг шалга:

- `app/(tabs)/soril.tsx` → 4 тоглоомын route, `app/leaderboard.tsx` (darkmode +
  i18n нэгдсэн — `showBadges`, `c.background` зөв эсэх).
- **Done:** 4 тоглоом + leaderboard light/dark хоёуланд ажиллана.

### B4. AI Buddy 3D avatar — Part 4 (feature)

`docs/AI_BUDDY_AVATAR_MESHY.md` зааврыг дагаж:

  review → `main`. `main` руу шууд push хийхгүй.
