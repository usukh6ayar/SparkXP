# Gamification

Core Rewards

- XP
- Gems
- Streaks
- Achievements

Reward feedback should include:

- Animation
- Glow
- Scale effect

# Accessibility

Minimum touch area:

44x44

Text contrast:

WCAG AA

Never use light gray text on white backgrounds.

---

# Motion Design

Duration

200-300ms

Use:

- Scale
- Fade
- Slide

Avoid:

- Bounce overload
- Excessive rotation

Motion should feel premium.
